    1. 在添加/删除程序中彻底删除sql server
    2. 将没有删除的sql server目录也删除掉
    3. 打开注册表编辑器，在HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager中找到PendingFileRenameOperations项目，并删除它。这样就可以清除安装暂挂项目
