windows10е’ҢUbuntuеҸҢзі»з»ҹпјҢж—¶й—ҙдёҚеҜ№

2016-11-10
tech

еҺҹеҚҡж–Ү

windows10е’Ңubuntu16.04еҸҢзі»з»ҹпјҢд»Қз„¶еҮәзҺ°дәҶе–ңй—»д№җи§Ғзҡ„иҖҒй—®йўҳпјҢиЈ…е®ҢеҗҺпјҢеңЁwindowsдёӢж—¶еҢәдёҚеҜ№пјҢд№ӢеүҚзҡ„иҖҒеҠһжі•жҳҜпјҡ

sudo gedit /etc/default/rcS
utc=yes ж”№жҲҗ utc=no

з„¶иҖҢе№¶жІЎжңүд»Җд№Ҳж•ҲжһңпјҢubuntu16.04йҮҢйқўж №жң¬е°ұжІЎжңүutcиҝҷдёҖйЎ№пјҒ
еҗҺжқҘи°·жӯҢдәҶеҚҠеӨ©жүҫеҲ°дәҶдёҖдёӘе®ҢзҫҺи§ЈеҶіеҠһжі•пјҡ
е…ҲеңЁubuntuдёӢжӣҙж–°дёҖдёӢж—¶й—ҙпјҢзЎ®дҝқж—¶й—ҙж— иҜҜпјҡ

sudo apt-get install ntpdate
sudo ntpdate time.windows.com

з„¶еҗҺе°Ҷж—¶й—ҙжӣҙж–°еҲ°зЎ¬д»¶дёҠпјҡ

sudo hwclock --localtime --systohc

йҮҚж–°иҝӣе…Ҙwindows10пјҢеҸ‘зҺ°ж—¶й—ҙжҒўеӨҚжӯЈеёёдәҶпјҒ

ж–°еҚҡж–Ү

дҪҶжҳҜеңЁubuntu24.04дёӯпјҢжӣҙж–°пјҡ

з®ҖеҚ•жқҘиҜҙпјҢеңЁUbuntu24.04дёӯпјҢhwclockе·Із»Ҹиў«ж·ҳжұ°дәҶпјҢе№¶йҮҮз”ЁдәҶtimedatectlеҺ»д»Јжӣҝе®ғгҖӮ еӣ жӯӨжҲ‘д»¬еҸҜд»ҘеңЁиҒ”зҪ‘зҠ¶жҖҒдёӯдҪҝз”ЁдёӢеҲ—д»Јз Ғе®һзҺ°UbuntuдёҺWindowsзҡ„ж—¶й—ҙеҗҢжӯҘгҖӮ

sudo timedatectl set-local-rtc 1
