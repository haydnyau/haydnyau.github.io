ubuntu2022е№ҙж— жі•ж·»еҠ еј•еҜјwindowsпјҢе®ҳж–№зҰҒз”ЁOS Prober

2022-01-15
tech
ubuntu  windows  octopress3.0

еҺҹеҚҡпјҡOS Prober иў«зҰҒз”ЁпјҢUbuntu жӯЈеңЁиҖғиҷ‘ж–°зҡ„еӨҡйҮҚеј•еҜјж–№жЎҲ

GNU GRUBпјҲGRUBпјүжҳҜдёҖдёӘжқҘиҮӘ GNU йЎ№зӣ®зҡ„еҗҜеҠЁеј•еҜјзЁӢеәҸгҖӮе®ғе…Ғи®ёз”ЁжҲ·еңЁи®Ўз®—жңәеҶ…еҗҢж—¶жӢҘжңүеӨҡдёӘж“ҚдҪңзі»з»ҹпјҢе№¶еңЁи®Ўз®—жңәеҗҜеҠЁж—¶йҖүжӢ©еёҢжңӣиҝҗиЎҢзҡ„ж“ҚдҪңзі»з»ҹгҖӮ

д»Ҡе№ҙ 6 жңҲпјҢз”ұдәҺдёҠжёё GRUB еј•еҜјзЁӢеәҸеңЁеҸ‘еёғ 2.06 зүҲжң¬ж—¶зҰҒз”Ё OS_Prober еҗҺжүҖеёҰжқҘзҡ„дёҖдәӣеҸҳеҢ–пјҢUbuntu зҡ„ејҖеҸ‘иҖ…жӯЈеңЁиҖғиҷ‘еҰӮдҪ•еңЁ Ubuntu 22.04 LTS дёӯз®ЎзҗҶеҸҢеј•еҜј/еӨҡеј•еҜјж–№жЎҲгҖӮ

Ubuntu дҫқйқ  OS-Prober жқҘжЈҖжҹҘе…¶д»–е·Іе®үиЈ…зҡ„ж“ҚдҪңзі»з»ҹпјҢд»ҘдҫҝиғҪеӨҹж–№дҫҝең°еңЁ GRUB еј•еҜјзЁӢеәҸдёӯжҳҫзӨәе®ғд»¬гҖӮиҝҷж„Ҹе‘ізқҖйҡҸзқҖ Ubuntu 22.04 иҪ¬еҗ‘ GRUB 2.06пјҢйқһ UEFI BIOS з”ЁжҲ·дёҚиғҪеҶҚеҗҜеҠЁд»»дҪ•е…¶д»–ж“ҚдҪңзі»з»ҹпјҢиҖҢеҹәдәҺ UEFI зҡ„зі»з»ҹеҰӮжһңйҖҡиҝҮ UEFI еј•еҜјзЁӢеәҸпјҢеҲҷд»Қз„¶еҸҜд»ҘеҗҜеҠЁе…¶д»–ж“ҚдҪңзі»з»ҹгҖӮиҝҷдёӘй—®йўҳдё»иҰҒжҳҜдјҡйҳ»зўҚйӮЈдәӣд»Қз„¶иҝҗиЎҢдј з»ҹ BIOS еҗҜеҠЁзЁӢеәҸзҡ„з”ЁжҲ·дҪҝз”Ёе·Іе®үиЈ…зҡ„зі»з»ҹгҖӮ

й»ҳи®ӨзҰҒз”ЁOS-Proberж—ЁеңЁйҖҡиҝҮж“ҚдҪңзі»з»ҹжЈҖжөӢеҠҹиғҪи§ЈеҶіжҪңеңЁзҡ„е®үе…Ёй—®йўҳпјҢеӣ дёәе®ғеңЁжЈҖжҹҘе…¶д»–ж“ҚдҪңзі»з»ҹж—¶пјҢдјҡжЈҖжҹҘе№¶жҢӮиҪҪзі»з»ҹдёҠзҡ„жүҖжңүеҲҶеҢәпјҢеҰӮжһңеҲ©з”Ёж–Үд»¶зі»з»ҹзҡ„жјҸжҙһпјҢеҗҜз”ЁиҜҘеҠҹиғҪдјҡеј•еҸ‘дёҖдёӘиғҪиў«й»‘е®ўеҲ©з”Ёзҡ„е®үе…Ёй—®йўҳгҖӮ

еӣ жӯӨпјҢйқўеҜ№е®үе…Ёе’Ңз”ЁжҲ·дҪ“йӘҢиҝҷдёӘдёӨйҡҫзҡ„йҖүжӢ©дёӢпјҢUbuntu ејҖеҸ‘иҖ…д№ҹдёҚеҫ—дёҚиҖғиҷ‘жҳҜеҗҰиҰҒйҮҚж–°еҗҜз”Ё OS-ProberгҖӮ

еӣ дёә OS_Prober еӯҳеңЁжҪңеңЁе®үе…ЁйЈҺйҷ©пјҢжүҖд»ҘйҮҚж–°еҗҜз”Ёе®ғе№¶дёҚжҳҜдёҖдёӘзҗҶжғізҡ„и§ЈеҶіж–№жЎҲпјҢзӣ®еүҚ Ubuntu еӣўйҳҹеҶ…йғЁд№ҹеңЁд»”з»ҶжҖқиҖғиҝҷдёӘй—®йўҳпјҢдҪҶдёҚжҺ’йҷӨиҝҷдёӘеҸҜиғҪжҖ§гҖӮ

Ubuntu еӣўйҳҹиҖғиҷ‘зҡ„е…¶д»–и§ЈеҶіж–№жЎҲеҢ…жӢ¬пјҡеҲӣе»әдёҖдёӘж–°зҡ„ GRUB жЁЎеқ—пјҢиҜҘжЁЎеқ—еҸӘеңЁз”ЁжҲ·жҹҘжүҫзі»з»ҹж—¶жүҚжү§иЎҢ OS-Prober зҡ„дёҖдәӣеҠҹиғҪпјҢз”ЁдәҺеўһеҠ дёҖдёӘй“ҫејҸеҗҜеҠЁйҖүйЎ№дё“й—ЁеӨ„зҗҶз”ЁжҲ·йңҖиҰҒеҗҜеҠЁ Windows зҡ„йңҖжұӮпјӣдәҰжҲ–жҳҜд»…еңЁе®үиЈ…зі»з»ҹж—¶иҝҗиЎҢ OS-ProberпјҢз„¶еҗҺдҫқйқ иҜҘдҝЎжҒҜиҝӣиЎҢеҗҺз»ӯеҗҜеҠЁпјӣжңҖеҗҺдёҖз§ҚеҸҜиғҪжҖ§жҳҜпјҢд»Һд»ҘеүҚзҡ„ Ubuntu зүҲжң¬еҚҮзә§еҲ° Ubuntu 22.04 ж—¶еҗҜз”Ё OS-ProberгҖӮ

жҲ‘д»¬иҝҳдёҚжё…жҘҡ Ubuntu еӣўйҳҹеңЁдёҠиҝ°йҖүжӢ©дёӯжңҖз»ҲдјҡеҰӮдҪ•иҝӣиЎҢжҠүжӢ©пјҢеңЁ Ubuntu 22.04 жӯЈејҸжҺЁеҮәд№ӢеүҚиҝҳжңүе……и¶ізҡ„иҖғиҷ‘ж—¶й—ҙпјҢеҸҜд»Ҙи®ҝй—® ubuntu-devel дәҶи§ЈжӣҙеӨҡиҜҰжғ…гҖӮжҲ‘д»¬д№ҹе°Ҷе…іжіЁ GRUB дёҠжёёеңЁжӯӨжңҹй—ҙжҳҜеҗҰд№ҹдјҡжҺЁеҮәзӣёеә”зҡ„и§ЈеҶіж–№жЎҲгҖӮ
